# 🎯 Number Guessing Game

Welcome to the **Number Guessing Game**, a fun and beginner-friendly Python project where your goal is to guess a randomly generated number between **1 and 100**. After each guess, the game provides a hint to help you get closer to the correct answer.

This project demonstrates the fundamentals of Python programming, including loops, conditional statements, user input, random number generation, and basic game logic.

---

## ✨ Features

* 🎲 Random number generation between **1 and 100**
* 💡 Helpful hints after every guess

  * **Higher number please**
  * **Lower number please**
* 🏆 Displays the total number of attempts
* 💻 Simple command-line interface
* 🚀 Beginner-friendly Python project

---

## 🛠️ Technologies Used

* Python 3
* Random Module

---

## ▶️ How to Play

1. Run the program.
2. Enter a number between **1 and 100**.
3. Follow the hints to adjust your next guess.
4. Keep guessing until you find the correct number.
5. The game will display how many attempts you took to win.
